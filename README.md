# claude-plugins

Personal marketplace of Claude Code plugins.

## Add this marketplace

```
/plugin marketplace add joshua-burnell-1/claude-plugins
```

## Available plugins

| Plugin | Description | Source |
|---|---|---|
| `prompt-squeeze` | Measures every prompt, surfaces avoidable spend, reports org-wide. Skill + UserPromptSubmit hook + MCP rollup with cited dollar/Wh/CO2e receipts. | [joshua-burnell-1/prompt-squeeze](https://github.com/joshua-burnell-1/prompt-squeeze) |

## Install a plugin

```
/plugin install prompt-squeeze@joshua-burnell-1
```
